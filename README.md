# 🛡️ Security Vulnerability Checklist

A beginner-friendly Python project that performs a **basic security assessment** by checking common security vulnerabilities such as weak passwords, outdated software, and unsafe user practices. It generates a simple security report with recommendations to help users improve their cybersecurity awareness.

---

## 📖 Project Overview

The **Security Vulnerability Checklist** is a command-line application built in Python. It asks users a series of security-related questions and evaluates their responses to identify potential risks.

This project is designed for students and beginners who want to learn the fundamentals of cybersecurity, risk assessment, and Python programming.

---

## ✨ Features

* 🔐 Checks password strength
* 💻 Verifies operating system update status
* 🛡️ Detects antivirus availability
* 🔑 Checks if Two-Factor Authentication (2FA) is enabled
* ⚠️ Identifies unsafe user practices
* 📊 Calculates a security score
* 📋 Generates a security report with recommendations

---

## 🛠️ Technologies Used

* Command Line Interface (CLI)

## 🚀 Getting Started

### Prerequisites

* Python 3.x installed on your computer

### Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/Security-Vulnerability-Checklist.git
```

2. Navigate to the project folder

```bash
cd Security-Vulnerability-Checklist
```

3. Run the program

```bash
python security_checklist.py
```
